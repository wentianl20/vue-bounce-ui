# vue-bounce-ui

---

#### 基于Vue3.0的轻量、可靠的移动端弹簧橡皮筋效果，兼容微信内置浏览器

## 安装

---

#### 通过 `npm` 安装:

```
npm i vue-bounce-ui
```

### 快速上手

---

```
import { createApp } from 'vue'
import App from './App.vue'
import SliderContainer from 'vue-bounce-ui'
import 'vue-bounce-ui/lib/vue-bounce-ui.css'

const app = createApp();

// 注册组件
app.use(SliderContainer);
```

### 浏览器支持

---

#### 与Vue3.0 一致
